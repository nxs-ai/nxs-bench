# NXS-Bench External Contributions

This file is maintained by `nxs-bench/.github/workflows/contribution-tracker.yml`.
It tracks merged pull requests from external contributors that count toward
NXS-Bench Advisory Board formation. The formation threshold is **10 external contributions**.

## Threshold Status

- External contributions counted: **0 / 10**
- Remaining until Advisory Board formation can proceed: **10**
- Advisory Board status: **Draft governance only until the threshold is met**
- Trigger message at threshold: `Threshold reached — Advisory Board formation can proceed`

## Counting Rules

- Count merged pull requests from authors whose `author_association` is not
  `OWNER`, `MEMBER`, or `COLLABORATOR`.
- Ignore bot accounts.
- Classify each counted contribution as `new scenario`, `bug fix`, or
  `methodology feedback`.
- Use this tracker together with [`governance/CHARTER.md`](governance/CHARTER.md)
  and [`governance/MEMBERS.md`](governance/MEMBERS.md).
