# Advisory Board Minutes

Publish one markdown file per quarterly meeting using the pattern
`YYYY-QN.md`.

## Required sections

- Date and attendees
- Agenda
- Conflict of interest disclosures and recusals
- Decisions taken
- Rationale for each decision
- Follow-up actions and owners

## Async votes

When a decision is made through a GitHub Issue between meetings, add a short
entry to the next quarterly minutes file with:

- issue link
- voting outcome
- quorum confirmation
- public rationale
